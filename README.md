# Company Profile Kelompok – Praktikum Git & GitHub

Website Company Profile (Compro) sederhana untuk tugas praktikum. Dibangun tanpa framework, hanya HTML, CSS, dan JavaScript vanilla dengan desain responsif dan struktur rapi untuk kolaborasi.


## Cara Menjalankan

- Buka file `index.html` langsung di browser, atau
- Jalankan server statis lokal apa pun (contoh dengan Python): `python -m http.server` lalu buka `http://localhost:8000/`.

## Pembagian Tugas (untuk GitHub)

- index.html → Dechellya Assyya (Ketua)
- about.html → Muhammad Fakhri N

- contact.html → Agil Hardiaman

## Workflow Git

- Setiap anggota bekerja pada branch masing-masing.
- Ajukan perubahan melalui Pull Request ke repository.
- Ketua melakukan review dan merge.
- Hindari push langsung ke `main`.

## Teknologi dan Prinsip

- HTML semantik
- CSS Flexbox dan Grid, mobile-first, media query: 768px dan 1024px
- JavaScript vanilla untuk hamburger toggle, smooth scroll, dan validasi form sederhana
