Ringkasan Proyek

Proyek Website Company Profile Mahasiswa ini merupakan inisiatif kolaboratif yang bertujuan untuk menampilkan identitas profesional dan hasil karya mahasiswa secara digital. Website ini dirancang sebagai portofolio bersama yang mencerminkan kompetensi teknis dan kreativitas tim.

Tujuan Pembuatan Website

  • Representasi Digital: Menyediakan platform resmi untuk memperkenalkan profil mahasiswa kepada dunia luar.
  • Portofolio Interaktif: Menampilkan proyek-proyek yang telah dikerjakan dengan tampilan yang menarik dan responsif.
  • Pusat Informasi: Memudahkan pihak eksternal untuk mendapatkan informasi kontak dan layanan yang ditawarkan.

Konsep Kolaborasi Git

Dalam pengembangan website ini, kami menerapkan konsep kolaborasi menggunakan Git. Setiap anggota tim bekerja pada bagian tertentu (feature branching), melakukan sinkronisasi melalui repositori GitHub, dan memastikan kualitas kode melalui proses Pull Request serta Code Review. Hal ini memungkinkan kami untuk bekerja secara paralel tanpa mengganggu pekerjaan satu sama lain.
