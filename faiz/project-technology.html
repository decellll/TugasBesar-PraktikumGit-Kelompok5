Teknologi yang Digunakan

Dalam membangun proyek ini, kami memanfaatkan stack teknologi web modern yang efisien:

HTML5

Berfungsi sebagai kerangka dasar (skeleton) website, memastikan struktur konten yang semantik dan aksesibel.

CSS3

Digunakan untuk penataan visual, tata letak (layout), dan memberikan estetika yang menarik melalui desain responsif.

JavaScript

Memberikan elemen interaktif pada website, seperti validasi form dan efek animasi sederhana untuk meningkatkan UX.

Git

Sistem Version Control yang kami gunakan untuk melacak setiap perubahan kode dan mempermudah kolaborasi antar anggota tim.

GitHub

Platform hosting repositori yang memfasilitasi manajemen proyek, penyimpanan kode di cloud, dan integrasi antar pengembang.
