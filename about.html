Tentang Kelompok

Kami adalah kelompok mahasiswa yang membangun website sederhana untuk mempraktikkan kolaborasi Git & GitHub, tanpa framework, namun tetap modern dan responsif.

Visi

Menguasai alur kerja kolaborasi modern melalui proyek nyata yang rapi, terstruktur, dan mudah dikembangkan.

Misi

  • Menerapkan semantic HTML, Flexbox, dan Grid.
  • Menulis CSS mobile-first dengan media query terukur.
  • Mengelola branch dan pull request secara disiplin.

Latar Belakang

Website ini dirancang sebagai Company Profile kelompok dalam tugas Praktikum Git & GitHub. Tujuannya agar setiap anggota berkontribusi dengan jelas melalui pembagian file yang tegas.

Struktur Jelas

Memudahkan branching dan review.

Standar Kode

Konsisten, ringkas, dan rapi.
