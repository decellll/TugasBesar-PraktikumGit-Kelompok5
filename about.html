Tentang Kelompok

Kami adalah kelompok mahasiswa praktikum yang mengembangkan website Company Profile untuk mempraktikkan kolaborasi Git & GitHub secara terstruktur, mulai dari branching hingga review kode.

Secara akademik, proyek ini membantu kami memahami hubungan antara struktur folder, pembagian tugas, dan workflow version control sehingga setiap perubahan dapat ditelusuri, dievaluasi, dan didokumentasikan dengan jelas.

Visi

Menjadi kelompok praktikum yang mampu menerapkan Git & GitHub secara disiplin dalam proyek web nyata, dengan kode yang rapi, mudah dipelihara, dan terdokumentasi.

Misi

  • Menerapkan semantic HTML, Flexbox, dan Grid untuk antarmuka yang responsif.
  • Mengelola branch, commit, dan pull request dengan alur yang terdokumentasi.
  • Melatih komunikasi teknis melalui review kode dan diskusi di GitHub.
  • Menjaga konsistensi standar penulisan kode di seluruh halaman.

Latar Belakang

Website ini dikembangkan sebagai Company Profile kelompok pada Praktikum Git & GitHub. Setiap anggota memegang tanggung jawab halaman tertentu, namun tetap berkolaborasi dalam satu repository yang sama.
