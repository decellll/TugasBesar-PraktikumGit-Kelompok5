Ringkasan Proyek

Website Company Profile kelompok yang dibangun tanpa framework, menggunakan HTML, CSS, dan JavaScript vanilla untuk kebutuhan minimal seperti hamburger menu, smooth scroll, dan validasi form.

Fitur

  • Desain responsif mobile-first.
  • Navbar sticky dengan hamburger menu.
  • Team grid dinamis: 1/2/3 kolom.
  • Smooth scroll ke anchor.
  • Form validasi sederhana di halaman Contact.

Tujuan Pembelajaran

  • Memahami workflow Git: branch, PR, review, merge.
  • Menerapkan struktur folder kolaboratif.
  • Menulis kode rapi dan semantik.

Teknologi

HTML

Struktur semantik dan aksesibel.

CSS

Flexbox, Grid, dan media query.

JS

Interaksi minimal, tanpa framework.
